RoastMaster Stripboard Layout Guide

Complete build reference for the RoastMaster control panel interface board.
Board connects the Raspberry Pi (via T-Cobbler Plus) to the MCP3008 ADC, PCA9685 PWM driver, and all panel controls (switches, pots, buttons, encoder).

1. Board Specifications

Stripboard

80 × 120 mm (31 rows × 47 columns)
Copper strips run horizontally (left to right)
Standard 0.1" (2.54 mm) hole pitch

T-Cobbler Plus

Adafruit #2028 — 40-pin GPIO breakout
Placed at cols 22-23, rows 3-22
20 track cuts required (1 per pin pair)

MCP3008

8-channel 10-bit SPI ADC (DIP-16)
Placed at cols 8-11, rows 10-17
In 16-pin DIP socket

PCA9685

16-channel 12-bit PWM driver (breakout board)
Connected via 6-pin header at col 35, rows 10-15
Breakout board mounted separately

Orientation convention: All row/column references assume you are looking at the component side (top) of the board. Row 1 is at the top, column 1 is at the left. Copper strips are on the bottom, running left to right. Track cuts are made from the copper side (flip the board — remember that left/right are mirrored).

4b. MCP3008 ADC (DIP-16 in socket)

4c. PCA9685 Header (6-pin, vertical)

The PCA9685 breakout board is mounted separately (on a standoff or the enclosure wall). A 6-pin header on the stripboard at col 35, rows 10-15 provides the I2C + power connection. Use a right-angle header or straight header with DuPont jumper wires to the breakout.

8. Off-Board Wiring

All panel-mount components (switches, pots, buttons, encoder, LEDs, meters) connect to the stripboard via hookup wire. Each external wire is soldered to a pad on the appropriate GPIO strip. Use the strip segment beyond the T-Cobbler pins (i.e., away from the board center) for clean terminal pads.

8a. Toggle Switches (SPST, active LOW)

Each toggle: one terminal to the GPIO pad, other terminal to GND. Switches pull the GPIO LOW when ON (closed). Pull-up resistors on the board hold the GPIO HIGH when the switch is OFF (open).

Shared GND pads: Multiple switches can share the same GND pad since they all connect to the same GND bus. Just use the nearest convenient GND strip.

Each pot has 3 pins: outer-1 → 3.3V, wiper (center) → MCP3008 channel, outer-2 → GND. The wiper connects to the MCP3008 channel input on segment A (cols 1-9) of the MCP3008 rows.

9. Power Wiring

Meanwell RT-65B to Stripboard

The Meanwell RT-65B has a +5V output (8A). Run this to the Pi's 5V GPIO pin via the stripboard.

Meanwell OutputBoard PadConnects To
+5VRow 3, Col 45T-Cobbler 5V strip (pins 2/4) — powers the Pi
COM (GND)Row 5, Col 45T-Cobbler GND strip (pin 6)
Power safety:
  • Powering the Pi via GPIO 5V pins bypasses the USB-C port's polyfuse protection. Ensure the Meanwell 5V output is clean and well-regulated (it is — the RT-65B is a quality supply).
  • Add an inline 3A fuse on the 5V wire for extra protection.
  • Do NOT put a physical switch in the 5V line to the Pi. Use the GPIO23 POWER toggle as a soft shutdown request only.
  • The Meanwell RT-65B also has +12V and -12V outputs — leave these unconnected unless you add 12V panel LEDs later.

Power Distribution on Board

                  ├── T-Cobbler Pin 2 (5V) at col 23 ──► Powers Raspberry Pi
                  ├── T-Cobbler Pin 4 (5V) at row 4, col 23 ──► 5V available

Pi 3.3V regulator ──► T-Cobbler Pin 1 (3V3) at row 3, col 22 ──► 3.3V left strip
                       ├── Jumper to Row 1 (primary 3.3V bus)

10. Build Sequence

Follow this order for the smoothest build. Test at each stage before proceeding.

  1. Prepare the board
    • Cut stripboard to 31×47 holes if needed
    • Mark component positions with a pencil on the component side
    • Make ALL 43 track cuts from the copper side (use craft knife + 3mm drill bit)
    • Verify every cut with multimeter continuity mode
  2. Power bus wires
    • Solder bus wires for rows 1, 2, 23, 24 (3.3V and GND buses)
    • Add bus jumper wires (section 7a)
    • Verify: row 1 has 3.3V from T-Cobbler, row 2 has GND, etc.
  3. T-Cobbler Plus
    • Solder the T-Cobbler header pins (cols 22-23, rows 3-22)
    • Connect ribbon cable to Pi and verify power/ground on all expected strips
  4. MCP3008 socket & wiring
    • Solder 16-pin DIP socket (cols 8+11, rows 10-17)
    • Solder decoupling cap (row 10-12, col 13)
    • Solder VDD/VREF and AGND/DGND jumper bridges
    • Solder SPI jumper wires (section 7b)
    • Solder MCP3008 power wires (section 7c)
    • Insert MCP3008 into socket (pin 1 dot toward row 10)
    • Verify: ls /dev/spidev0.0 on Pi, then test with 1 pot
  5. Pull-up resistors & debounce caps
    • Solder all 14 pull-up resistors (section 5)
    • Solder all 10 capacitors (section 6)
  6. Test switches & buttons
    • Wire up 1 toggle switch and 1 button temporarily
    • Verify GPIO reads change when pressed/toggled
    • Wire remaining switches and buttons
  7. PCA9685 header & wiring
    • Solder 6-pin header at col 35, rows 10-15
    • Solder I2C and power jumper wires (sections 7d, 7e)
    • Connect PCA9685 breakout via DuPont wires
    • Verify: i2cdetect -y 1 shows 0x40
    • Test with 1 VU meter
  8. Final assembly
    • Wire all pots, remaining buttons, encoder
    • Wire VU meters with calibrated series resistors
    • Wire LEDs with series resistors
    • Run full software test
